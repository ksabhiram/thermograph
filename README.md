# Thermograph
# 🌡️ Interactive temperature and pressure visualisation — straight from your data to your screen.
ThermoGraph is a browser-based interactive plotting tool for scientists, engineers to Plot thermal and pressure data v/s timeadd the blocks, tune the view, and get publication-ready charts in seconds — no installation required.
